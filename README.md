# Project-7
AI Study Buddy
AI Study Buddy is a browser-based study assistant that helps users turn pasted notes into summaries, flashcards, quizzes, and quick answers using local text analysis.

***Features***

Smart note summarization

Key term extraction

Auto-generated flashcards

Quiz question generation

Ask questions based on pasted notes

Sample notes loader

Works fully in the browser

***Tech Stack***

HTML

CSS

JavaScript

Local NLP-style text scoring

No backend or API required
